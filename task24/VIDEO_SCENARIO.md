## 0:40–2:00 | Показываем код (rag_with_citations.py)

Открыть файл, показать 3 ключевых места:

**1. SYSTEM_PROMPT с шаблоном трёх секций**
```python
SYSTEM_PROMPT = """...
ОТВЕТ: <ответ со ссылками [1], [2]>
ИСТОЧНИКИ: [N] Название (chunk_id=X, score=Y)
ЦИТАТЫ: [N] «цитата из материала»
..."""
```
> «Три секции — обязательные. Модель не может ответить без источников, это не просьба, это структура.»

**2. Порог "не знаю"**
```python
NOT_KNOW_THRESHOLD = 0.50
if max_score < threshold:
    return {"not_know": True, "answer": "Не знаю..."}
```
> «Если максимальный cosine similarity ниже 0.5 — не даём модели фантазировать. Честный ответ лучше уверенной чепухи.»

**3. Двойной детектор**
```python
model_said_not_know = "не знаю" in answer_final.lower()
```
> «Второй слой: даже если score прошёл порог, но модель сама понял что контекст не отвечает — фиксируем это.»
